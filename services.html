Specialized in

HTML

HTML adalah bahasa pemrograman atau pengkodean yang digunakan untuk membuat halaman web. HTM merupakan singkatan dari Hypertext Markup Language.

CSS

CSS adalah singkatan dari Cascading Style Sheets, yaitu bahasa pemrograman yang digunakan untuk menentukan tampilan dan format website. CSS berfungsi untuk mengatur gaya visual elemen pada halaman web, seperti warna, font, ukuran, dan jarak.

JS

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web yang interaktif dan dinamis. JavaScript merupakan salah satu teknologi inti dari World Wide Web.
